# Churn Data Analysis 📊❌

Este projeto consiste em um script Python voltado para a Análise Exploratória de Dados (EDA) de cancelamentos de clientes (Churn). O objetivo é identificar padrões, comportamentos e anomalias na base de dados que possam justificar a perda de clientes, gerando insights visuais para a tomada de decisões estratégicas.

---

## 🚀 Tecnologias e Bibliotecas

* **Python 3**
* **Pandas**: Limpeza de dados (remoção de valores nulos e colunas irrelevantes) e análise estatística descritiva (frequência absoluta e relativa).
* **Plotly Express**: Geração automatizada de gráficos dinâmicos e histogramas interativos com mapeamento de comportamento.
